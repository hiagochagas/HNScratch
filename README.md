# HNScratch

HNScratch is a ObjectiveC framework that makes it easier to make scratchable views and it works both in ObjectiveC or Swift projects.

## Example

![](https://github.com/hiagochagas/HNScratch/blob/main/example.gif)

To run the example project, clone the repo, and run `pod install` from the Example directory first.

## Requirements
* iOS 14.5+
* Swift 5
* Objective-C

## Installation

HNScratch is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'HNScratch'
```
